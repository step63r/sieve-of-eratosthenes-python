# sieve-of-eratosthenes-python

## Description

List all primzahlen less than or equal to the specified integer.

## Requirement

- Python 3.9.7

## Usage

Run script as below.

```
$ python ./main.py 20
[2, 3, 5, 7, 11, 13, 17, 19]
```
